# BlazeMap

High-performance mapping infrastructure for modern applications.

## What we build

- **Offline-first maps** — Vector tiles that work without internet
- **Geocoding** — Fast address search and reverse geocoding
- **Routing** — Optimized path finding for vehicles and pedestrians
- **Self-hosted** — Run on your infrastructure, own your data

## Tech stack

Built with Rust for performance and reliability.
